Ghostlight in Browser

How the agent's activity looks. Governance lives in the binary.

Agent activity effects

The phantom cursor, click ripples, scan-lines, and the rest while the agent works. Turn off for a completely quiet browser.

Action captions

A subtitle naming each action ("Reading page", "Screenshot"). Handy when recording a demo; off for everyday.

Governance and policy

Which domains the agent may touch, capability limits, and the audit log are configured in the Ghostlight binary you installed -- not here. Run ghostlight config or ghostlight policy. This extension holds no policy; it only executes what the binary allows.
