## Why

Ghostlight already has the hard strategic ingredients: a distinctive user-context job, visible
delight, strong model ergonomics, candid non-goals, a real free core, current packages, verifiable
release artifacts, and unusually deep trust documentation. It does not need more generic launch
copy. It needs the last mile to become one coherent public truth.

The exercise found that the public project page still says v0.5.7 and that Linux live verification
is owed, while the repository and latest release say v0.6.0 and the README says Windows/Linux are
verified. The extension still requires the informed manual path while store review is open. GitHub
community intake lacks issue and PR templates. These defects are repairable, but borrowed
attention should not be used to discover them.
